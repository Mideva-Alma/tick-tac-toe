# Prompt-Powered Kickstart: Tic Tac Toe Web App

This project is a beginner-friendly toolkit for building a browser-based Tic Tac Toe game using vanilla JavaScript, HTML, and CSS. It demonstrates dynamic grid rendering, multiple difficulty levels, and a level-up system.

## Quick Start
1. Clone or download this repository.
2. Open the project folder in your code editor.
3. Open `index.html` in your browser to play the game.

## Features
- Beginner (3x3), Medium (4x4), and Hard (6x6) grid options
- Player vs Player and Player vs Computer modes
- Win tracking and level-up system

## Requirements
- Any modern browser (Chrome, Firefox, Edge, etc.)
- No installation or server required

## File Structure
- `index.html` – Main HTML file
- `src/style.css` – Styling for the game
- `src/script.js` – Game logic and interactivity
