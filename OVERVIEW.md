# Tic Tac Toe Web App Overview

## Features
- 3x3 grid game board (black box design)
- Play against another person or the computer (robot)
- Tracks and displays number of wins for each player
- Simple, clean UI

## Technologies Used
- HTML, CSS, JavaScript (runs in browser)

## File Structure
- index.html: Main HTML file
- style.css: Styling for the game
- script.js: Game logic and interactivity

## How It Works
- User selects mode: Player vs Player or Player vs Computer
- Players take turns marking X or O
- Game checks for win/draw after each move
- Win counters update and display after each game

## How to Run
Open `index.html` in your browser. No server or Python required.
